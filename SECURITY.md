# Security Policy

## Reporting a Vulnerability

If you find a security vulnerability in Thinking Canvas, please report it privately rather than opening a public issue.

- Email: tjdev23@gmail.com
- Or use GitHub's private vulnerability reporting: https://github.com/interfacedreams/thinking-canvas/security/advisories/new

Please include a description of the issue, steps to reproduce, and any relevant details about your environment. You can expect an acknowledgment within a few days.

## Supported Versions

Only the latest release is supported with security fixes.
